# Questões resumo 

### *Revisão de Ponteiros e Alocação Dinâmica de Memória*

- Como alocar dinamicamente um vetor de inteiros e preenchê-lo com valores sequenciais?
- Como trocar os valores de duas variáveis usando ponteiros?

### *Complexidade de Algoritmos*

- O que acontece com o tempo de execução de um algoritmo O(n) se o tamanho da entrada dobrar?
- Qual é a complexidade de um algoritmo com dois loops aninhados que percorrem uma matriz n x n?

### *Listas Lineares de Dados*

- Como implementar uma lista linear usando um vetor e realizar operações de inserção, remoção e busca?
- Como remover elementos duplicados de uma lista linear?

### *Lista Simplesmente Encadeada*

- Como implementar uma lista simplesmente encadeada e inserir elementos no início, no fim e em uma posição específica?
- Como inverter a ordem dos nós em uma lista encadeada?

### *Lista Circular*

- Como implementar uma lista circular e inserir elementos no início e no fim?
- Como verificar se uma lista encadeada é circular?

### *Lista Duplamente Encadeada e Multilista*

- Como implementar uma lista duplamente encadeada e inserir elementos no início e no fim?
- Como imprimir os elementos de uma lista duplamente encadeada em ordem reversa?

### *Nó Cabeça em Listas Encadeadas*

- Como implementar uma lista simplesmente encadeada com nó cabeça e inserir elementos no início e no fim?
- Como remover um nó com um valor específico de uma lista com nó cabeça?

### *Nó Cabeça em Listas Circulares*

- Como implementar uma lista circular com nó cabeça e inserir elementos no início e no fim?
- Como verificar se uma lista circular com nó cabeça está vazia?

### *Nó Cabeça em Listas Duplamente Encadeadas*

- Como implementar uma lista duplamente encadeada com nó cabeça e inserir elementos no início e no fim?
- Como remover um nó com um valor específico de uma lista duplamente encadeada com nó cabeça?

### *Pilha e Fila*

- Como implementar uma pilha usando um vetor e realizar operações de push e pop?
- Como implementar uma fila usando uma lista encadeada e realizar operações de enqueue e dequeue?
